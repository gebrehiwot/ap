Encapsulation

Protecting the internal state. In Advanced Java, we use this to ensure data integrity.

public class Account {
    private double balance; 
    public void deposit(double amt) {
        if(amt > 0) balance += amt; // Business Logic Protection
    }
}

Abstraction

Define the "What", hide the "How". Essential for microservices and API design.

interface Database {
    void connect(); // Implementation is hidden
}
