📦 1. class & object

start here Class = blueprint · Object = instance in memory.

📐 class

class Student { String name; int id; void study() { System.out.println(name + " studies"); } }

🧩 objects

Student s1 = new Student(); s1.name = "Alice"; Student s2 = new Student(); s2.name = "Bob";
🔒 2. encapsulation

private fields, public methods — protect data integrity.

public class BankAccount { private double balance; public void deposit(double amt) { if(amt > 0) balance += amt; } public double getBalance() { return balance; } }
🌿 3. inheritance

extends — child gets parent members. "is‑a" relationship.

parent

class Animal { void eat() { System.out.println("eating"); } }

child

class Dog extends Animal { void bark() { } } // Dog has eat() + bark()

super() calls parent constructor.

class Vehicle { String brand; Vehicle(String b) { brand = b; } } class Car extends Vehicle { Car(String b) { super(b); } }
🌀 4. polymorphism

many forms — overloading (compile‑time) & overriding (runtime).

overloading

class Calc { int add(int a, int b) { return a+b; } int add(int a, int b, int c) { return a+b+c; } }

overriding

class Cat extends Animal { @Override void eat() { System.out.println("cat eats"); } }
Animal[] pets = {new Dog(), new Cat()}; for (Animal a : pets) a.eat(); // dynamic dispatch
📄 5. interface

contract — what a class must do. implements

interface Flyable { void fly(); } class Bird implements Flyable { public void fly() { System.out.println("bird soars"); } }

multiple interfaces

interface Swimmer { void swim(); } interface Runner { void run(); } class Athlete implements Swimmer, Runner { public void swim() { } public void run() { } }
🎭 6. abstract class

partial implementation — can have abstract + concrete methods. cannot instantiate.

abstract class Shape { abstract double area(); void display() { System.out.println("display shape"); } } class Circle extends Shape { double radius; double area() { return 3.14 * radius * radius; } }
⚠️ 7. exceptions

handle errors — try, catch, finally, throws, throw.

checked

try { Thread.sleep(1000); } catch (InterruptedException e) { e.printStackTrace(); }

unchecked

int x = 10 / 0; // ArithmeticException
try { int[] nums = {1,2}; System.out.println(nums[5]); } catch (ArrayIndexOutOfBoundsException e) { System.out.println("out of bounds!"); } finally { System.out.println("always runs"); }
🚀 FINAL PROJECT: Zoo Management System

brings everything together

🐘 ZooApp – classes, encapsulation, inheritance, interface, polymorphism, exceptions

// --- interfaces --- interface Feedable { void feed(String food) throws Exception; } interface Trainable { void performTrick(); } // --- abstract base class --- abstract class Animal { private String name; // encapsulation private int age; public Animal(String name, int age) { this.name = name; this.age = age; } public String getName() { return name; } public abstract void makeSound(); // abstract public void sleep() { // concrete System.out.println(name + " is sleeping"); } } // --- inheritance + interfaces --- class Lion extends Animal implements Feedable, Trainable { public Lion(String name, int age) { super(name, age); } public void makeSound() { System.out.println(getName() + " roars!"); } public void feed(String food) throws Exception { if (!food.equals("meat")) { throw new Exception("Lions only eat meat!"); } System.out.println(getName() + " eats " + food); } public void performTrick() { System.out.println(getName() + " jumps through hoop"); } } class Penguin extends Animal implements Feedable { public Penguin(String name, int age) { super(name, age); } public void makeSound() { System.out.println(getName() + " honks!"); } public void feed(String food) { System.out.println(getName() + " eats " + food); } } // --- main simulation (polymorphism + exceptions) --- public class Zoo { public static void main(String[] args) { Animal[] animals = { new Lion("Simba", 5), new Penguin("Pingu", 3) }; for (Animal a : animals) { a.makeSound(); // polymorphic call a.sleep(); if (a instanceof Feedable) { try { ((Feedable) a).feed("fish"); // Lion throws exception } catch (Exception e) { System.out.println("Error: " + e.getMessage()); } } if (a instanceof Trainable) { ((Trainable) a).performTrick(); } System.out.println("---"); } } }

✅ demonstrates: class/object, encapsulation (private name), inheritance (Lion/Penguin extends Animal), interface (Feedable/Trainable), polymorphism (Animal array + makeSound()), exceptions (Lion feed throws).
