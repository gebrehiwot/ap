Pillar 01

Encapsulation

Goal: State Security

Protect object integrity by hiding data and exposing only "safe" pathways. 
public class Account {
    private double balance;

    public void deposit(double amt) {
        if(amt > 0) balance += amt;
    }
}
Pillar 02

Abstraction

interface CloudStorage {
    void save(String data);
}

class AWS implements CloudStorage {
    public void save(String data) { 
        // Complex logic hidden 
    }
}

Goal: Complexity Hiding

Define the contract (interface), hide the implementation.
Pillar 03

Polymorphism

Goal: Dynamic Flexibility

The ability to treat different objects as their parent type. The JVM decides the method at Runtime.

List<Payment> list = new ArrayList<>();
list.add(new PayPal());
list.add(new CreditCard());

for(Payment p : list) p.process(); // DYNAMIC DISPATCH
