Explain how these 4 pillars work together in a single Java file:

// ABSTRACTION: The generic idea
abstract class BankAccount {
    private double balance; // ENCAPSULATION

    public BankAccount(double amount) { this.balance = amount; }
    
    public double getBalance() { return balance; }
    public abstract void applyBonus(); 
}

// INHERITANCE
class StudentAccount extends BankAccount {
    public StudentAccount(double b) { super(b); }

    @Override // POLYMORPHISM
    public void applyBonus() {
        System.out.println("No fees for students!");
    }
}
