Encapsulation State Security

Moving beyond simple "getters/setters," teach your students about Object Integrity. An object must protect its internal state from invalid data.

public class BankAccount {
    private double balance; 

    // Guard Clause pattern
    public void withdraw(double amount) {
        if (amount > balance) {
            throw new InsufficientFundsException();
        }
        this.balance -= amount;
    }
}

Inheritance Code Reusability

Explain the "Is-A" relationship. In Advanced Java, we prefer Composition (Has-A) over deep inheritance trees to avoid the "Fragile Base Class" problem.

Abstraction Contract Design

Abstraction is about defining a contract. Using interface tells the world what your system can do without exposing how it does it.

public interface CloudStorage {
    void upload(File file);
    void download(String id);
}

// implementation is hidden!

Polymorphism Dynamic Dispatch

The crown jewel of OOP. It allows the JVM to decide which method implementation to run at runtime based on the actual object type.
